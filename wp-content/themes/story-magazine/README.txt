== Description ==
Story Magazine is elegant magazine WordPress theme developed specifically for personal, streamlined blog websites. This theme is easy-to-use so you will focus on enjoying your blog. Story Magazine brings fully responsive layout to your site, theme supports the awesome Theme Customizer for all theme settings and enables to create your own color scheme.  Memorable and deterministic magazine design makes the difference.
For more information about Story Magazine please go to https://wpmasters.org/downloads/story-magazine-theme/.


== Installation ==
1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Story Magazine in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
